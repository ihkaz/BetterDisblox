# Examples

Example scripts are stored in the repository under `examples/`.

Every example loads BetterDisblox from the public bundle:

```lua
local BetterDisblox = loadstring(game:HttpGet("https://raw.githubusercontent.com/ihkaz/BetterDisblox/main/dist/BetterDisblox.lua"))()
```

## Files

- `examples/basic.lua`
- `examples/slash-commands.lua`
- `examples/buttons.lua`
- `examples/select-menu.lua`
- `examples/modals.lua`
- `examples/components-v2.lua`
- `examples/webhook.lua`
- `examples/webhook-components-v2.lua`
- `examples/rest.lua`
- `examples/command-registry.lua`

## Recommended Start

Start with `examples/basic.lua`, then test one feature at a time.

Do not paste a real bot token into public scripts, screenshots, or GitHub commits.
